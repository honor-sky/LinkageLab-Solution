# LinkageLab - 카카오 접근성 이슈 해결 과제 
* 보안 이슈를 고려해 보고서가 작성된 문서 링크는 따로 첨부하지 않습니다.
(updated date. 2024.11.07)
</br>

## Intro
- 카카오 서비스에서 발생한 해결되지 못한 접근성 이슈을 해결

## Tech Stack
- Android, JetPack
- Kotlin, XML

- 각 이슈별로 개선된 화면 및 기능 개발
